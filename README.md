# App

This is a project for mobile terminal.

## Introduction

### 测试环境

*  安装Node、npm ;
*  npm install ;
*  Set port 80 ,  /node_modules/browser-sync/lib/default-config.js ; 

```
    /**
     * @property port
     * @type Number
     * @default 3000
     */
    port: 80 ,
```
 
### 开发自动化工具
*  start run.bat ;

### 测试环境
*  cd App/wap/
*  npm run lite 
*  or http-server -a 0.0.0.0 -p 80

### 测试数据
*  /Data
*  Mockjs 模拟
