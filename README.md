# App

This is a project for mobile terminal.

## Introduction

### 测试环境

*  安装Node、npm；
*  npm install；
*  Set port 80，/node_modules/browser-sync/lib/default-config.js；

```
    /**
     * @property port
     * @type Number
     * @default 3000
     */
    port: 80,
```
 
### 自动化工具

*  cd wap；
*  npm install；
*  start run.bat；
